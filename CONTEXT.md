# Contexto del proyecto AVA

## ¿Qué es AVA?
AVA (Alfabetización Visual Argentina) es una aplicación web educativa para personas sordas no alfabetizadas de Argentina. Utiliza Lengua de Señas Argentina (LSA) como puente hacia la lectoescritura.

## Problema que resuelve
Personas sordas que no saben leer ni escribir, o que solo reconocen palabras de memoria. El enfoque es visual y sin depender del texto para enseñar texto.

## Usuario principal
Personas sordas no alfabetizadas de Argentina. El caso de referencia es un joven de 21 años que usa WhatsApp, TikTok e Instagram.

## Flujo de aprendizaje (MVP)
1. El usuario ve un video en LSA de una palabra
2. Ve la palabra escrita asociada al video
3. Practica escribiéndola en un input
4. Recibe feedback visual inmediato

## Vocabulario del MVP (Etapa 1)
- Saludos y respuestas básicas: hola, chau, sí, no, bueno, gracias, por favor, perdón
- Preguntas interrogativas: qué, cómo, dónde, cuándo, quién, por qué
- Números del 1 al 10
- Colores básicos

## Stack tecnológico
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Base de datos: PostgreSQL
- Deploy: Vercel (frontend) + Railway (backend)

## Estado actual
Sprint 0 en curso. Proyecto inicializado con React + Vite + Tailwind CSS.

## Principios de diseño
- Interfaz 100% visual, sin texto para navegar
- Inspirado en TikTok e Instagram (video vertical, interacción simple)
- Feedback inmediato y visual
- Mobile-first
